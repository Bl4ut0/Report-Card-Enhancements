# Shared Wrapper Files

These files are the version-neutral foundation for the Warcraft Logs compatibility layer.

| File | Purpose |
|---|---|
| `WCL_Compat.gs` | Public wrapper API and credential-mode detection. |
| `WCL_V1.gs` | V1 REST implementation helpers. |
| `WCL_V2.gs` | V2 OAuth/GraphQL implementation helpers. |

Replacement sets may include copies of these files when users need a self-contained folder for their exact CLA/RPB version.
